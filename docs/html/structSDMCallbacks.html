Detailed Description

Collection of callback functions for SDM_Init() and EComPort_Init()

Both f_nSRSTStageX take a void* context parameter. When invoked with the reference libraries, the context parameter contains the CSAPBCOM handle to facilitate the system reset, if necessary.
